# 🍳 EC Each Chef - 智能菜谱助手

一个基于Gradio的智能菜谱搜索应用，支持菜谱分类浏览、食材筛选、智能推荐、成本计算等功能。

## ✨ 功能特点

- 📖 **菜谱分类浏览** - 中餐、西餐、特色菜分类查看
- 🧅 **食材筛选** - 根据现有食材智能推荐菜谱
- 🔧 **工具筛选** - 根据现有厨具推荐菜谱
- ✨ **智能推荐** - 基于用户偏好的个性化推荐
- 💰 **成本计算** - 自动计算每道菜的食材成本
- 🔥 **热量计算** - 每道菜的热量分析
- 📝 **菜谱收藏** - 收藏喜欢的菜谱
- ✅ **烹饪记录** - 记录做过哪些菜
- 📅 **时令推荐** - 根据季节推荐时令菜谱
- ➕ **新增菜谱** - 支持用户自定义添加菜谱
